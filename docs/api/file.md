# 云盘管理模块

云盘管理模块（`file`）提供完整的云盘文件管理功能，包括文件上传、下载、管理和分享。

## 模块说明

::: tip 💡 云盘 vs 图床
- **云盘**（`file`）：用于存储各种类型的文件，支持完整的文件管理操作
- **图床**（`image`）：专门用于图片存储，提供图片链接和图片处理功能

两者是独立的存储空间，互不影响。
:::

## 子模块

云盘模块包含以下子模块：

- `sdk.file.upload` - 文件上传功能
- `sdk.file.share` - 分享管理功能
- `sdk.file.*` - 其他文件管理方法（列表、下载、移动、删除等）

查看完整的云盘操作示例，请访问 [云盘操作示例](/examples/file)。
